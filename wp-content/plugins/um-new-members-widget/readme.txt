== Description ==
Add-on to Ultimate member to show latest new members in a widget or with shortcode.
To use the widget, simply add the widget in Wordpress to your sidebar. To use the shortcode, add the shortcode umnm-new-members to a page.

Need any changes, other features? Post a support item, and if I like it, I'll add it.

Like this plugin? I appreciate a review :) For more information see [really-simple-plugins.com/um-new-members](https://really-simple-plugins.com/um-new-members/).

For more Ultimate Member add-ons, see [really-simple-plugins.com/download/category/ultimate-member/](https://really-simple-plugins.com/download/category/ultimate-member/).

== Installation ==
To install this plugin:

1. Download the plugin
2. Upload the plugin to the wp-content/plugins directory,
3. Go to “plugins” in your WordPress admin, then click activate.
4. In appearance, widgets, drag the New Members widget to a sidebar, or add the shortcode [umnm-new-members number=5] to one of your pages.

== Frequently Asked Questions ==
= Is there a shortcode? =
* Yes, you can use [umnm-new-members number=5]. You can put this shortcode anywhere on your site.

= Can I override the template for the widget? =
* Yes. Copy the template in templates/new-members.php to:
theme-folder/ultimate-member/templates/um-members-widgets/new-members.php, and edit to your liking. The plugin will pick it up automatically
